# 農の里

## 概要

「農の里」は、自然農・自然栽培・有機農法などに関心のある人、地域の農家・団体が、イベント、学び、非公開の栽培記録、在来種、農具の貸し借りを通してつながる地域コミュニティアプリです。現在はSupabaseを使ったPhase 2プロトタイプまで進んでいます。

## スマホで開くURL

https://livingcolour.github.io/nou-no-sato/

## Phase 1の目的

バックエンド実装の前に、画面構成、主要導線、GUIの雰囲気を確認できる状態にすることを目的としています。中心価値は、地元の仲間を見つけること、イベントにつながること、農法をやさしく学ぶこと、栽培記録を基本非公開で残すこと、在来種・固定種への関心を高めることです。

## 画面一覧

- ホーム
- 地元の仲間を探す
- イベントを見る
- 農具を借りる・貸す
- 農法を学ぶ
- 栽培記録を残す
- 在来種マップを見る
- マイページ簡易イメージ
- 農家・団体・農具の管理画面

農具シェアのDBを既存Supabaseへ追加するSQLは `supabase/migrations/20260713_add_farmers_and_equipment_share.sql` です。

## ローカル確認方法

このリポジトリ直下で簡易HTTPサーバーを起動します。

```bash
python -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000/index.html
```
