# 農の里

## 概要

「農の里」は、自然農・自然栽培・有機農法などに関心のある人が、地元でゆるくつながり、学び、畑の記録を残し、リアルイベントへ参加できる地域コミュニティアプリのPhase 1静的UIプロトタイプです。

## スマホで開くURL

https://livingcolour.github.io/nou-no-sato/

## Phase 1の目的

バックエンド実装の前に、画面構成、主要導線、GUIの雰囲気を確認できる状態にすることを目的としています。中心価値は、地元の仲間を見つけること、イベントにつながること、農法をやさしく学ぶこと、畑ノートを基本非公開で残すこと、在来種・固定種への関心を高めることです。

## 画面一覧

- ホーム
- 地元の仲間を探す
- イベントを見る
- 農法を学ぶ
- 畑ノートを記録する
- 在来種マップを見る
- マイページ簡易イメージ

## ローカル確認方法

このリポジトリ直下で簡易HTTPサーバーを起動します。

```bash
python -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000/index.html
```
