# 農の里 現在状態

## プロジェクト名

農の里

## アプリの目的

自然農・自然栽培・有機農法・菌ちゃん農法などに関心のある人が、地元でゆるく仲間を見つけ、リアルイベントに参加し、農法を学び、自分の畑の記録を残し、在来種・固定種への関心を高めるための地域コミュニティアプリです。

## Phase 1の位置づけ

Phase 1は、バックエンド実装前に画面構成、主要導線、UIの雰囲気、プロダクト方針を確認するための静的UIプロトタイプです。

現時点ではDB、API、ログイン、認証、投稿保存などの本番機能はありません。

## 現在の実装状態

- 静的HTML/CSS/JavaScriptで構成されています。
- `index.html` はアプリシェルで、hash routingにより複数画面を静的に切り替えます。
- `data/mock-data.js` に画面表示用のダミーデータがあります。
- `js/main.js` がルーティングと各画面の描画を担当します。
- ホーム、仲間一覧（個人＋団体）、団体ページ、イベント一覧/詳細、農法一覧/詳細、畑ノート一覧/作成、在来種マップ/詳細、在来種の情報提供、団体向け管理（メニュー/プロフィール/イベント登録）、マイページを表示できます。
- 画像素材は外部コピーではなく、`assets/visuals/hero-satoyama.jpg` とCSS表現で構成しています。
- スマホ向け下部ナビがあります。

## コミュニティ活性化のための実装（2026-06-27 追加）

「発見→最初の接触→参加→定着→還元」のループを埋めるため、以下を追加しました。バックエンドは持たない方針を維持しています。ただし「気になる／活動を受け取る／イベントに誘う」の3つの軽い操作状態だけは、ブラウザの localStorage（キー `nounosato:ui`、内部セット名 `interested` / `following` / `invited`）に保存し、再訪でも残します（`js/main.js` の `loadUi` / `saveUi`）。フィルタの選択状態はセッション内のみ。栽培記録・プロフィール・各フォームの入力は引き続き保存しません。

- 最初の一歩を小さく：イベントの「気になる」トグル（一覧/詳細/マイページで連動）と、ホームの「はじめての方へ」3ステップ導線。
- 個人ピアの可視化：個人データ `peers` を新設し、仲間ページを「近くの個人」＋「地域の団体・サークル」の2層に再構成。匿名サマリー帯（個人◯人・団体◯団体＋プライバシー注記）を表示。
- 継続関係：団体に活動リズム、季節の便り（一方向の読み取り専用お知らせ）、「活動を受け取る」トグル。イベントに「参加した人の声」。
- 歓迎の温度：守りの注記と並走する歓迎メッセージ（見学OK・途中参加OKなど）。
- 軽い貢献導線：在来種の「情報提供（運営確認つき）」フォーム（自由投稿ではなくmoderated submission）。掲載フローは `docs/nou-no-sato-decisions.md` を参照。
- 相互リンク：在来種↔団体↔イベントを双方向リンク。
- フィルタの機能化：仲間（農法）/イベント（種別）のチップを実際に動く `button` 化（`aria-pressed`、モバイルは横スクロール）。
