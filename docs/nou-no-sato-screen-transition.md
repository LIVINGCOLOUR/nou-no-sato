# 農の里 Phase 1 画面遷移設計

## 1. この文書の目的

この文書は、「農の里」Phase 1静的UIプロトタイプの次の検討材料として、画面構成、主要導線、画面遷移、実装優先度を整理するための設計メモです。

実装指示ではなく、画面を増やす前に「どの画面を最初に作るか」「どの導線を同一画面内で代替できるか」「旧設計書から何をFuture扱いにするか」を判断しやすくするための文書です。

## 3. ユーザーの基本導線

### はじめて見る

ユーザーはホームで、「農の里」が自然な栽培に関心のある人のための地域コミュニティであることを理解します。最初の選択肢は、仲間、イベント、農法、畑ノート、在来種です。

### 地元の仲間を探す

ホームから仲間一覧へ進み、市町村程度の地域、関心のある農法、活動状況を見ます。初期実装では独立したプロフィール詳細画面を作らず、一覧カードの展開で関心、活動範囲、参加しやすいイベントへの導線を確認します。正確な住所や畑の場所は表示しません。

### イベントを見る

ホームまたは仲間カードからイベント一覧へ進み、運営登録イベントを確認します。詳細では、日時、地域、内容、定員、参加前に知ることを確認します。Phase 1では誰でもイベント作成できる導線は作りません。

### 農法を学ぶ

ホームから農法一覧へ進み、自然農、自然栽培、有機農法、菌ちゃん農法などを優劣ではなく考え方の違いとして眺めます。詳細では、特徴、向き合う視点、注意点、出典確認の方針を読みます。

### 畑ノートを記録する

ホームまたはマイページから畑ノート一覧へ進みます。記録作成・編集では、日付、作物、写真、ひとことを残します。Phase 1では基本非公開で、位置情報や詳細な畑住所を扱わない前提です。初期実装では独立した詳細画面を急がず、一覧カードと作成・編集画面で確認できる範囲に留めます。

### 在来種を知る

ホームから在来種マップへ進み、地域と在来種・固定種の関係を見ます。詳細では、作物名、地域、背景、特徴、出典確認の考え方を確認します。正確な採種場所や個人宅は表示しません。

### マイページで振り返る

マイページでは、自分の関心、畑ノート数、参加予定イベント、お気に入りの農法や在来種を簡単に振り返ります。詳細なSNSプロフィールではなく、個人情報を出しすぎない確認画面として扱います。公開範囲の説明は、初期は設定画面ではなくマイページ内カードで代替します。

## 4. 実装優先度

### A. 最初に作る画面: Phase 1 Core

最初のクリック可能プロトタイプでは、中心価値に直結する以下の画面を優先します。

- S01 ホーム
- S02 仲間を探す一覧
- S04 イベント一覧
- S05 イベント詳細
- S06 農法を学ぶ一覧
- S07 農法詳細

- S09 畑ノート作成・編集
- S11 在来種マップ
- S12 在来種詳細
- S13 マイページ

### B. 初期は同一画面内で代替する画面: Inline / Card Expand

初期実装では独立画面にせず、既存画面内のカード展開や説明カードで代替します。

- S03 仲間プロフィール詳細
  - 理由: Phase 1ではDMがないため、独立詳細画面より一覧カード展開で十分です。
- S10 畑ノート詳細
  - 理由: 初期は一覧カードと作成・編集画面で記録内容の確認を代替できます。
- S14 設定・公開範囲説明
  - 理由: 初期はマイページ内の説明カードで、公開範囲と安心方針を伝えられます。

以下はPhase 1主導線から外し、利用確認後に必要性、安全性、運営負担を見て再検討します。

- 困りごとから探す

- Q&A

自由DMではなく、公開範囲を控えたプロフィール確認とイベント参加を主導線にします。S03は初期の独立画面にせず、一覧内のカード展開で代替します。

### イベント導線

- S04 イベント一覧 → S05 イベント詳細
- S05 イベント詳細 → S13 マイページ

イベントは運営登録を前提にし、誰でも作成できる導線はPhase 1では作りません。

### 農法学習導線

- S06 農法を学ぶ一覧 → S07 農法詳細

学びを読んで終わらせず、観察や記録へ戻る導線を持たせます。農法比較は優劣ではなく、考え方や向き合い方の違いとして表現します。

### 畑ノート導線

- S08 畑ノート一覧 → S09 畑ノート作成・編集

- S08 畑ノート一覧 → S13 マイページ

畑ノートは基本非公開です。公開タイムライン、他人の畑記録閲覧、位置情報共有はPhase 1では扱いません。S10は初期の独立画面にせず、一覧カードと作成・編集画面で代替します。

- S11 在来種マップ → S12 在来種詳細
- S12 在来種詳細 → S11 在来種マップ

在来種はユーザー投稿地図ではなく、運営整備データを前提にした学習・関心喚起の導線です。

### マイページ導線

- S13 マイページ → S08 畑ノート一覧
- S13 マイページ → S04 イベント一覧
- S13 マイページ → S14相当の公開範囲説明カード

マイページはSNS的な自己表現ではなく、自分の学びと活動を整理する入口にします。S14は初期の独立画面にせず、マイページ内の説明カードで代替します。

## 7. Phase 1から外す旧導線

### 困りごとから探す / 条件付き相談

旧設計書では初心者の入口として有効でしたが、現在の「農の里」では地域の仲間・イベント・学びが中心価値です。困りごと診断や相談導線を前面に置くと、相談アプリや診断アプリに見えやすいため、Phase 1主導線から外します。

### Q&A

旧設計書では質問・回答が重要機能でしたが、現在のPhase 1では公開Q&A掲示板を作りません。回答の正誤、農法間の対立、運営負担が増えやすいため、まずは運営登録イベントと中立的な学習コンテンツに寄せます。

### ブロック・通報・冷却制御

これらはQ&Aや公開コミュニケーションを作る場合に必要な安全機能です。Phase 1ではDM、チャット、Q&Aを作らないため、画面遷移の主導線から外します。将来Q&Aを扱う場合は、安全機能とセットで再検討します。

### 季節チャレンジ / 仮想畑

継続を促す機能や遊び要素は魅力がありますが、Phase 1の中心価値をぼかしやすいため、まずは仲間、イベント、学び、畑ノート、在来種の導線を固めた後に検討します。
