# 農の里 Phase 1 UI Prototype

## 目的

「農の里」は、自然農・自然栽培・有機農法などに関心のある人が、地元でゆるくつながり、学び、畑の記録を残し、リアルイベントへ参加できる地域コミュニティアプリです。

このフォルダはバックエンド実装前に、画面構成、導線、GUIの雰囲気を確認するための静的プロトタイプです。

## Phase 1 の対象ユーザー

- 家庭菜園を始めたい初心者
- 自然農、自然栽培、有機農法、菌ちゃん農法などに興味がある人
- 周囲に相談相手や仲間が少ない人
- 小さく実践していて、地元の仲間やイベントを探したい人

## Phase 1 の機能範囲

- 地元の仲間を市町村程度の地域と関心ごとで探す
- 運営登録イベントを見る
- 農法の違いをやさしく比較する
- 畑ノートを基本非公開で記録する画面イメージ
- 在来種・固定種を地域別カードと地図風UIで見る
- マイページの簡易イメージを見る

## Phase 1 でやらないこと

- ユーザー登録
- 投稿保存
- DB設計
- API実装
- AI相談
- Q&A掲示板中心の体験
- リアルタイムチャット
- DM
- アバター
- 仮想広場
- ランキング
- ポイント課金
- アイテム課金
- 農産物販売
- SNS投稿

## 画面一覧

- ホーム
- 地元の仲間を探す
- イベントを見る
- 農法を学ぶ
- 畑ノートを記録する
- 在来種マップを見る
- マイページ簡易イメージ

## デザイン方針

- 緑、白、淡い生成り、少量のオレンジを基調にする
- 山、畑、空を想起する明るい配色にする
- スマホファーストで下部ナビを用意する
- 白いカード、やわらかい影、丸みのあるボタンで安心感を出す
- 写真が入る前提の鮮やかな枠を用意しつつ、現時点ではCSSの仮ビジュアルで表現する
- 農法の優劣をつけず、地域差や実践条件の違いを大切にする
- 畑ノートは基本非公開として、人間関係トラブルを増やす機能を避ける
- 仲間探しでは本名、詳細住所、畑の正確な場所を前面に出さない
- 農法比較は「おすすめ順位」ではなく、条件や考え方の違いとして見せる

## 今後バックエンド化する場合の候補

- 運営管理のイベント登録と公開フロー
- ユーザーのプロフィール登録
- 畑ノートの非公開保存
- 農法コンテンツのCMS化
- 在来種データの運営確認済みデータベース化
- 市町村単位のゆるい検索
- 通知とお気に入り機能
- 管理者向けの通報・非表示・承認機能

## 参考画像から取り入れた要素

- 大きな自然ビジュアルを前提にしたヒーロー
- 緑を中心に、白と生成りで余白を取る色使い
- カード型レイアウト
- スマホアプリ画面を意識した下部ナビ
- アイコン付きの主要導線
- イベント、仲間、農法、畑ノート、在来種マップを並べる情報設計

## 参考画像から採用しなかった要素

- 設計書画像そのものの紙面再現
- 画像内の写真素材の切り出し
- AI相談を主役にする導線
- DM、リアルタイムチャット、ランキング、課金、販売機能
