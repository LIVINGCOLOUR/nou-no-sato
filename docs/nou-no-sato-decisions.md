# 農の里 決定事項

## 決定済みのプロダクト方針

- 「農の里」は、自然な栽培に関心のある人が地元でゆるくつながる地域コミュニティアプリです。
- 中心価値は、地元の仲間、リアルイベント、学び、畑ノート、在来種・固定種への関心です。
- AI相談アプリではありません。
- Q&A掲示板やSNS的な投稿拡散を中心にしません。
- 人間関係トラブルを増やしやすい機能はPhase 1では避けます。

## Phase 1で作るもの

- ホーム
- 地元の仲間を探す導線
- 運営登録イベントを見る導線
- 農法を学ぶ比較表示
- 畑ノートを記録する画面イメージ
- 在来種マップを見る画面イメージ
- マイページ簡易イメージ
- スマホファーストの下部ナビ
- 画面確認用のダミーデータ

## Phase 1で作らないもの

- 本格的なログイン
- ユーザー登録
- 投稿保存
- DB設計
- API実装
- AI相談
- Q&A掲示板中心の体験
- リアルタイムチャット
- DM
- アバター
- 仮想広場
- ランキング
- ポイント課金
- アイテム課金
- 農産物販売
- SNS投稿
- 複雑な権限管理

## 「やさいの背景」との違い

「やさいの背景」は、売場の野菜にQRを付け、野菜が育った背景動画を見せる体験型アプリです。

「農の里」は、自然農・自然栽培・有機農法などに関心のある人が、地元で仲間を見つけ、学び、記録し、リアルイベントへ参加できる地域コミュニティアプリです。

この2つは別アプリとして扱います。

## `kotoba` とは別リポジトリで管理する方針

- 「農の里」は `kotoba` とは完全に別プロジェクトです。
- 以前 `kotoba` リポジトリにPRを作成しましたが、mergeせずcloseしました。
- 今後の正しいリポジトリは `https://github.com/LIVINGCOLOUR/nou-no-sato` です。
- `kotoba` のブランチやPRを正本として扱わないでください。

## UI/UX方針

- 緑、白、淡い生成り、少量のオレンジを使います。
- 山、畑、空を想起する明るい配色にします。
- スマホファーストで設計します。
- 白いカード、やわらかい影、丸みのあるボタンで安心感を出します。
- 自然写真が入る前提の鮮やかな雰囲気にしますが、Phase 1ではCSSの仮ビジュアルで表現します。
- 初心者が安心して眺められる密度にします。
- 農業専門家向けの硬い業務UIにはしません。

## 個人情報・位置情報・畑ノート公開範囲の方針

- 仲間探しでは本名を前面に出しません。
- 詳細住所や畑の正確な場所は表示しません。
- 地域表示は市町村程度に留めます。
- 畑ノートは基本非公開です。
- 位置情報や詳細な畑住所を扱わない想定です。
- DMやリアルタイムチャットはPhase 1では入れません。

## 農法比較で優劣をつけない方針

- 自然農、自然栽培、有機農法、菌ちゃん農法、不耕起・草生栽培を優劣で並べません。
- 「これが唯一の正解」と見せません。
- 比較は、条件、考え方、続けやすさ、地域差を理解するためのものです。
- 農法ごとの実践条件や価値観の違いを尊重します。
